# EduRise Jumper

An educational vertical jumper game that highlights the importance of securing a child's academic journey from Primary School to Study Abroad.

## Tech Stack
- **Engine**: Vanilla JavaScript / HTML5 Canvas 2D
- **Styling**: TailwindCSS
- **Build Tool**: Vite 5
