# She Shield Protector

An interactive, high-fidelity mobile-optimized catcher/deflector game highlighting Bajaj Allianz Women's Critical Illness Cover ("SheShield").

## Product Theme & Concept
The game educates players about the unique health risks faced by women and the importance of financial protection.
- **Deflect Threats**: The player slides the SheShield to block critical illness threats (Stress, Heart Issue, Breast Cancer, Diabetes). Blocking threats awards score, symbolizing active protection.
- **Absorb Habits**: The player catches positive health practices (Exercise, Nutrients, Annual Check-up, Savings) to boost score and restore Shield Integrity.
- **Premium Cover Rider**: A rare diamond power-up that expands the shield size and adds a safety grid at the bottom, mimicking an insurance rider that offers complete protection.

## Tech Stack
- **Game Engine**: Vanilla JavaScript + HTML5 Canvas API (dynamic rendering, custom particles, screen shakes, collision physics).
- **Styling**: Tailwind CSS (loaded via CDN) for high-performance glassmorphic user interface overlays (menus, HUD, lead form).
- **Build/Server**: Vite 6.x
